# Mais-ligados-ao-agro-forte
projeto agrinho sobre agro forte e futuro sustentavel 

# Objetivos 
Fazer com que as pessoas entendam a importancia sobre agro forte e futuro sustentavel.

# Recusrsos tecnicos
# ⚙️Estrutura e Metadados (HTML5)
<!DOCTYPE html>: Declaração que define o documento como HTML5.
lang="pt-br": Atributo de acessibilidade e SEO que define o idioma principal da página como português do Brasil.
Tags <meta>: * charset="UTF-8": Garante a codificação correta de caracteres e acentos (como ç, á, õ).
name="viewport": Configuração essencial para garantir a responsividade (adaptabilidade do layout em telas de celulares e tablets).

# Estilização Avançada (CSS3)
Embora as regras estejam aplicadas dentro de uma tag <style> no próprio documento, o código utiliza propriedades modernas de CSS:
Manipulação de Fundo (Backgrounds):
Uso de imagem externa via url() com propriedades de preenchimento inteligente (background-size: cover e background-position: center).
background-attachment: fixed: Cria um efeito parallax simples, onde o fundo fica travado enquanto o conteúdo rola por cima.
rgba(0, 0, 0, 0.75): Uso de cores com canal Alpha para criar uma camada preta semitransparente sobre o fundo, melhorando o contraste e a leitura do texto.
Layout Moderno com Flexbox:
O seletor #telaEntrada utiliza display: flex.
As propriedades justify-content: center e align-items: center garantem o alinhamento e centralização perfeita dos elementos na tela (tanto na vertical quanto na horizontal).
flex-direction: column: Organiza os elementos filhos empilhados um embaixo do outro.
Design de Interface (UI):
Customização de botões (#telaEntrada button) usando cantos arredondados (border-radius: 10px), remoção de bordas padrão (border: none) e feedback visual de interatividade (cursor: pointer).
